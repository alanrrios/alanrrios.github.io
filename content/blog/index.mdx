## How this site is built

Under the hood, this is a lightweight docs-style site powered by Astro with content authored in MDX. Astro handles routing and page generation while MDX keeps each post in plain markdown with optional JSX components when needed.

Deployment runs on Cloudflare, which keeps hosting simple and globally fast. Styling is handled with Tailwind, and the content structure is organized so new journal entries can be added quickly without changing application code.

The stack is intentionally minimal:

Astro for static-first pages and layout composition
MDX for markdown-native writing with component flexibility
Cloudflare for deployment and edge delivery
